## 1. P0 — Fixed in 0.7.2

### P0-1 PTY injection in `inject_to_terminal` (`server.py:926-`)
- **Before**: user-supplied `content` written directly to `/dev/pts/X`; control sequences and multi-line payloads executed as terminal input.
- **Fix**: `_sanitize_pty_payload()` strips `\x00-\x08 \x0b-\x1f \x7f`, caps at 8 KB, rejects empty-after-strip.

### P0-2/3 Lock held across subprocess stdout (`server.py` `send_message_async` + `send_message`)
- **Before**: `with self._lock:` wrapped the entire `for line in proc.stdout:` loop, blocking every other thread holding `_lock` for the full Claude run.
- **Fix**: lock now only scopes `self.messages.append` + truncation; stdout iteration and `_parse_status` run without the lock.

### P0-4 Bridge `stop()` NameError (`feishu-bridge.py`)
- **Before**: `subprocess.run(...)` in `stop()` had no module-level import; local imports elsewhere were not visible here, so shutdown crashed and orphan cleanup never ran.
- **Fix**: `import subprocess` added at module top.

### P0-5 Dead code after `return False` (`server.py` `ClaudeSession.is_live`)
- **Before**: three init lines sat after `return False`, never executed; a refactor touching this tail could silently lose `_lock` initialisation.
- **Fix**: dead lines removed.

## 2. P1 — Open (delegated — see `GLM-FIX-BUNDLE.md`)

## 4. Cross-cutting architectural suggestions

1. **Bridge state machine**: collapse `_last_delivered_hash / _forward_baselines / _last_working_text / _prev_live / _response_cards / _session_index_map` into a single `SessionState` dataclass keyed by `session_id`, with a per-session lock. Makes the card-lifecycle invariant enforceable and gives a natural TTL hook.
2. **Atomic JSONL writes**: introduce `atomic_write(path, data)` helper using `tempfile.mkstemp` in same dir + `os.replace`. Replace every `r+ + truncate` writer for tasks.jsonl / sessions metadata / bridge persistence.
3. **Request body limit middleware**: wrap `BaseHTTPRequestHandler.rfile.read` to reject `content_length > 1 MiB` uniformly; removes 12 point fixes and forecloses the class.
4. **Doc-version hook**: `scripts/sync-version.sh` driven by pre-commit: single source `VERSION`, propagates to README / CLAUDE.md / DESIGN-DECISIONS / package.json.
5. **SessionConfig vs runtime split**: hardcoded IPs, cwd prefixes, and token paths should move to `config.json` with env-var override — currently 6 hardcoded magic strings in `server.py` + 4 in bridge.
